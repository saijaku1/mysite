値
100 ～ 900:
この範囲内で指定する必要があります。それ以外の値を設定すると、バグが発生する可能性があるので注意してください。
100: 最も細い。細字。
400: 通常の太さ（絶対値）。
500 ～ 800: だんだん太くなる。
900: 最も太い。
キーワード
数字を書くのが面倒なときは、以下のキーワードを使用できます：

normal: 通常の太さ。（400 と同じ）
bold: 太字。（700 と同じ）
bolder: 親要素より太くなる。
例: 親要素が 400 なら 700 になる。
lighter: 親要素より細くなる。
例: 親要素が 600 なら 400 以下になる。

ポイント
太さが極端に変わると、読みやすさに影響する場合があります。適切な太さを選びましょう。
normal や bold はシンプルな設定がしたいときに便利ですが、細かく調整したい場合は数値を使う方が正確です。値
100~900。これ以上大きくしたり小さくしたりするとバグが発生することがあるので、書く際には注意！

おさらい！

  • CSSで色を付けたい場合はcolor:色;と書きます！
  • CSSで大きさを変えたい場合はfont-size:数字pxなど;と書きます！
  • 文字の太さを付けたい場合はfont-weight:値;と書きます。
  • font-weightの値はpxなどの単位が必要なく、100~900と数字だけで書きます。※もしくはキーワードのbold、normalなどを使うといい。
