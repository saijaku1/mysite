  • HTMLのタグには開始タグ・終了タグの二つ表示があります。※終了タグがないものもある。
  • hタグはサイトの見出しを表示することができます。
  • pタグは文章を表示することができます。
  • brタグは文章の改行したいときに使うタグです。他のタグとは違い、終了タグは必要ありません。
