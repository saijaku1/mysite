そういえば、さっきタグが二つあったでしょ？
                
<h1>こんにちは！</h1>
最初のタグ(こんにちは！の左にあるタグ)のことを開始タグというんだ
                    
<h1>←これを開始タグという！
それから、最後のタグ(こんにちは！の後にあるタグ)のことを終了タグというんだ
                
</h1> ←この最後のタグを終了タグという！
終了タグはタグの<の後に/(スラッシュ)をつける必要があるんだ。
/(スラッシュ)の意味は〇〇タグを終わりにするよ！っという意味があるんだ！
結構出てくるから覚えたほうがいいよ！
それじゃあ次はpタグについてやっていこうか。
pタグとはサイトで言う説明や文字を作るんだよ!
                
<p>これはpタグの説明です！</p>
書き方はさっきのh1タグと一緒でタグ同士を囲むんだ！
実際に書いてみよう！
ブラウザでpタグを使って"今日はいい天気です。"と出力させてください。
いいね！pタグは他のタグと同じで開始タグと終了タグで囲むからわかりやすいよね！
それじゃあ、今度はそのbrタグについて学んでいこう！!
brは別名:改行タグと呼ばれるもので、文章を改行できるんだ。
brはbreakの省略。breakは「壊れる」、「破る」とかの意味もあるけど、実際は「改行する」っていう意味があるんだ！
brは他のタグとは少し違ってて、それは終了タグが必要ないんだ！
                
<br>←これでOK！
                
            
                
<br></br>←この終了タグは消さないとエラーになるよ！brタグを打つときは終了タグはなしって覚えとこう！
実際に打ってみよう!

pタグの中にある"プログラミングで"のあとにbrタグを入れてください。

できたかな？答えはこんな感じだよ！

おさらい！

  • HTMLのタグには開始タグ・終了タグの二つ表示があります。※終了タグがないものもある。
  • hタグはサイトの見出しやタイトルを表示することができます。
  • pタグは文章や説明を表示することができます。
  • brタグは文章の改行をしたいときに使うタグです。他のタグとは違い、終了タグは必要ありません。
  • わからないところがあったら、これまでのレッスンを見直しましょう。
