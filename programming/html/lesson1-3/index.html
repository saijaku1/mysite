  • 数字書きをしたい場合はolタグを使います
  • hタグはサイトの見出しやタイトルを表示することができます。
  • pタグは文章や説明を表示することができます。
  • brタグは文章の改行をしたいときに使うタグです。他のタグとは違い、終了タグは必要ありません。
